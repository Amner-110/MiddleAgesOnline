Middle Ages Online
==================

編集方法
_______

1.自分のブランチを作って作業を開始します。

http://i.imgur.com/2KqCsqA.jpg

2.完成したら自分のブランチのトップにあるPullRequestをクリック

http://i.imgur.com/Yk23Fxu.jpg

3.編集内容と説明を記載してRequestを送信

4.マージされるまでお待ちください。

・問題の報告、提案や要請はIssuesから
